Andy Pantry Static Site Draft Bundle

This folder contains the current draft site files:
- index.html
- support.html
- privacy.html
- styles.css
- app.js
- config.js

Before publishing, update config.js with:
- supportEmail
- homepageUrl
- chromeStoreUrl

Recommended local checks:
- npm run preflight:site
- npm run site:preview
- npm run site:open
